口袋大龙虾 Pocket Lobster

口袋大龙虾是一款运行在普通 Android 手机上面的双智能体原生 AI 助手应用。它把 OpenAI 旗下的编码智能体 Codex 与 2026 年以来最具代表性的全能私人 AI 智能体形态之一 OpenClaw 同时带到一台非 root 安卓手机上，并让它们共享安卓原生终端、完整 Ubuntu Linux 运行时和通过 Shizuku 打通的 system shell 三条执行链路。

项目当前已经具备以下核心能力：双智能体协作，双终端共享工作环境，Shizuku 授权下的系统级命令与 UI 自动化基础能力，共享存储的全局文件访问能力，多搜索源与网页自动化能力，MCP 与 skills 扩展能力，以及面向手机与无障碍场景重构的新聊天页面、权限管理、提示词管理、对话管理、模型管理和附件上传能力。

它不是单纯的聊天应用，而是一台真正装进口袋里的移动端 AI 工作站。两个智能体既可以处理软件开发与工程维护，也可以处理文档整理、批量文本、PDF、表格、多媒体转码、网页资料获取、文件处理与复杂任务交付。更进一步，口袋大龙虾已经具备自进化能力，智能体可以直接参与修改自己的源码、触发云端构建、产出新安装包并继续迭代。
