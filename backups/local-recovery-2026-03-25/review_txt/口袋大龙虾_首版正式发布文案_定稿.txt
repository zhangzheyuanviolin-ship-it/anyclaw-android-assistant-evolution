口袋大龙虾 Pocket Lobster 首版正式发布文案 定稿
生成日期：2026-03-25

口袋大龙虾正式进入首个稳定发布阶段。

这不是一个普通的手机聊天应用，也不是把桌面 AI 壳层简单搬到安卓上的移植项目。口袋大龙虾真正做成的，是把两款原本主要运行在桌面系统、终端环境或云端中的顶级智能体，第一次完整压缩进了一台普通的非 root 安卓手机，并让它们共享一套真实可执行、可交付、可持续演进的工作环境。

在这一个应用里，同时运行着两套智能体系统。第一套是 Codex，它是 OpenAI 旗下的编码智能体，也是当前最强的软件与代码智能体之一，擅长理解代码库、修改工程、调试问题、执行命令和推进复杂技术任务。第二套是 OpenClaw，它代表的是 2026 年以来真正把 AI 从传统聊天工具推进到全能私人助手形态的那一类智能体系统，能够调用工具、执行复杂任务、组织流程并交付最终成果。过去，这两类智能体往往分别活在桌面电脑、命令行终端或者云端环境里；而口袋大龙虾把它们真正带到了手机里。

更关键的是，口袋大龙虾不是只让它们“能聊天”，而是给它们配齐了三条真正可用的执行链路。第一条是安卓原生终端环境，第二条是完整的 Ubuntu Linux 发行版运行时，第三条是通过 Shizuku 授权打通的 system shell 系统链路。两个智能体都可以根据任务需要，自主在这些环境之间切换工作。它们既可以进入安卓原生终端处理本地任务，也可以进入完整 Ubuntu Linux 环境执行开发、脚本、编译和复杂命令；同时还可以通过 system shell 触达系统级命令、读取设备状态，并进一步具备 UI 自动化能力。正是这三条链路的打通，让 AI 在一台普通安卓手机上第一次真正获得了接近完整工作站的执行能力。

口袋大龙虾还有一个决定性的突破，就是它拿到了共享存储的全局文件访问能力。很多同类项目里的智能体最终都被困在自己的应用沙箱里，文件只能在私有目录里兜圈子，产出结果也很难直接交付给用户。口袋大龙虾则打通了 Manage External Storage 权限，让两个智能体都可以直接读取和处理手机共享存储中的文档、代码、图片、音频、视频、安装包和其他文件，再把结果直接交付回用户真正能继续使用的位置。这一点让它从“AI 在手机里跑起来了”，变成了“AI 真的可以在手机上替您做事并交付结果了”。

这也意味着，口袋大龙虾的用途远远不止软件开发。只要有双智能体、双终端、系统权限、共享文件访问和网络工具链，它就可以处理文档整理、表格与 PDF 处理、批量文本加工、多媒体转码与音视频处理、资料检索、网页抓取、工作流自动化、工程开发和设备维护等大量真实任务。它不是单一场景的工具，而是一台真正装进口袋里的通用 AI 工作站。

为了让这些能力在手机上真正可用，口袋大龙虾还围绕安卓场景做了专门的交互重构。新版大龙虾聊天页面简化了传统 OpenClaw 桌面前端的复杂配置，加入了更适合手机和无障碍使用的功能入口，包括权限管理中心、提示词管理、对话管理、模型管理以及附件上传能力。用户既可以手动完成模型配置、权限状态查看和提示词映射，也可以直接把需求交给两个智能体中的任意一个，让 AI 自己去完成这些配置。对于不熟悉终端命令的普通用户来说，这一步非常关键，因为它把原本需要技术背景才能完成的复杂操作，尽可能压缩成了手机端可以直接使用的入口。

与此同时，口袋大龙虾并不把智能体局限在本地环境里。当前环境已经配备了多搜索源检索、网页访问和 Web 自动化能力，支持百度、夸克、必应、谷歌、DuckDuckGo、Tavily 等多种搜索方式，也兼容 OpenClaw 的 skills 生态和 MCP 能力扩展。也就是说，两个智能体不只是能用手机本地资源，它们还能主动从互联网上获取资料、下载文件、抓取页面、补充工具能力，把手机扩展成一个面向现实世界持续工作的 AI 平台。

而口袋大龙虾最特别的一点，是它已经具备了自进化能力。只要给两个智能体中的任意一个配置好 GitHub 令牌，或者准备好安卓开发环境，它们就可以根据用户需求和任务需要，直接修改自己的源码、触发云端工作流、构建新安装包、下载产物、安装新版，并和用户一起继续测试、继续修复、继续演进。当前这个稳定版本，本身就是在真实手机环境里，通过一位完全不懂代码的视障用户与智能体长期协作、长期验证、长期迭代，最终一步一步发展出来的成果。

口袋大龙虾首个稳定版本真正证明了一件事情：只要 AI 拥有足够真实的工具环境、足够广泛的权限边界和足够完整的交付链路，那么一台小小的安卓手机，也可以成为真正的生产终端、开发终端、创作终端和个人 AI 工作平台。

这不是桌面体验的缩水版，而是移动端 AI 基础设施的一次真正突破。口袋大龙虾，把顶级编码智能体和顶级全能智能体一起带到了安卓手机上，并让它们第一次真正开始为现实任务交付成果。
